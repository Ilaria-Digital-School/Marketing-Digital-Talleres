# Miniproyecto : planificación de una sesión de Trello + Pomodoro

## Descripción del mini-proyecto Este mini-proyecto tiene como finalidad consolidar las competencias de organización personal del vendedor digital, integrando dos herramientas clave ya trabajadas en el curso: **Trello** (gestión de tareas y proyectos) y el **método Pomodoro** (gestión del tiempo y la productividad). El estudiante deberá diseñar, configurar y ejecutar una sesión de trabajo real utilizando ambas herramientas de forma combinada y coherente. El proyecto se inscribe en la continuidad de las actividades previas sobre organización personal del vendedor y el método Pomodoro, y permite aplicar de manera concreta lo aprendido en un contexto profesional simulado de marketing digital. ## Competencias a validar - Configurar y estructurar un tablero de Trello adaptado a un flujo de trabajo de marketing digital. - Aplicar el método Pomodoro para gestionar el tiempo de manera eficiente durante una sesión de trabajo. - Integrar ambas herramientas en una rutina de trabajo planificada y documentada. - Reflexionar sobre la propia organización y proponer mejoras basadas en la experiencia. ## Contexto profesional Como futuro experto en marketing digital, la capacidad de organizarse de forma autónoma y eficiente es fundamental. La combinación de Trello y Pomodoro permite estructurar las tareas de forma visual y gestionar el tiempo con disciplina, dos habilidades esenciales para implementar y monitorear estrategias de marketing digital en el día a día.
