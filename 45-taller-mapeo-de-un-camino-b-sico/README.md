# Taller: mapeo de un camino básico

RECORDATORIO TEÓRICO El mapeo del recorrido del cliente (customer journey map) es una representación visual que ilustra la experiencia de un usuario desde el primer punto de contacto con una marca hasta la conversión y, en algunos casos, la fidelización. Su objetivo es identificar cada etapa que atraviesa el usuario, las emociones que experimenta, los puntos de fricción y las oportunidades de mejora. Las etapas clásicas de un recorrido básico son: Descubrimiento, Consideración, Decisión, Compra y Post-compra. Para cada etapa se deben identificar: el canal utilizado (redes sociales, buscadores, email, etc.), las acciones del usuario, sus emociones y pensamientos, y los puntos de dolor (pain points). Modelo mental clave: el recorrido no es lineal. Los usuarios pueden retroceder, saltar etapas o abandonar el proceso. Un error frecuente en principiantes es construir un recorrido demasiado ideal, ignorando las fricciones reales. Otro error habitual es desconectar el recorrido de las personas (personas) previamente definidas, lo que produce un mapa genérico sin utilidad práctica. Mini-ejemplo: María, 34 años, emprendedora, descubre un software de gestión de proyectos a través de un artículo en LinkedIn (Descubrimiento). Compara planes y lee reseñas en Google (Consideración). Se suscribe a una prueba gratuita (Decisión). Completa el pago tras recibir un email de recordatorio (Compra).
