# Experto en marketing digital

Adquiere un dominio completo del marketing digital para desarrollar estrategias. Habilidades: implementar una estrategia de marketing web; organizar la aplicación operativa de una comunicación de marketing web; monitorear y evaluar los resultados de una comunicación de marketing web; comprender las necesidades de sus objetivos; conocer y saber cómo usar las herramientas de marketing web; configurar una estrategia de automatización de marketing; establecer una estrategia de automatización de marketing; analizar y optimizar el recorrido del usuario en los canales digitales; utilizar herramientas para medir y gestionar el rendimiento digital; configurar escenarios de automatización de marketing adaptados a los objetivos estratégicos;
