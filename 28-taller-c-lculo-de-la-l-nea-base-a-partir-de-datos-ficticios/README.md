# Taller: Cálculo de la línea base a partir de datos ficticios

**Recordatorio teórico** Antes de comenzar el taller, repasa estos conceptos esenciales. La línea base (o baseline) es el valor de referencia de un KPI en un momento dado, antes de cualquier acción de mejora. Sirve como punto de partida para medir el progreso real de una estrategia digital. Sin línea base, es imposible saber si un objetivo SMART es alcanzable o si las acciones implementadas están generando resultados. La línea base debe calcularse a partir de datos históricos fiables y representativos, generalmente de los últimos 30, 60 o 90 días según el KPI considerado. Para calcularla, se promedian los valores del período seleccionado, se identifican anomalías (picos o caídas excepcionales) y se decide si incluirlas o excluirlas del cálculo, justificando siempre la decisión. Errores frecuentes: usar un período demasiado corto (una semana no es representativa), ignorar la estacionalidad, o confundir la línea base con el mejor resultado histórico. La línea base no es el máximo alcanzado, sino el rendimiento habitual y realista del negocio. Mini-ejemplo: si una tienda online registró tasas de conversión del 2,1 %, 1,9 %, 2,3 %, 2,0 % y 2,2 % durante los últimos cinco meses, su línea base es la media = (2,1+1,9+2,3+2,0+2,2)/5 = 2,1 %.
