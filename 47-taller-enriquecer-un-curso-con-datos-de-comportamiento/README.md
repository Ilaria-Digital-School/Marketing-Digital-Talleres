# Taller: enriquecer un curso con datos de comportamiento

RECORDATORIO TEÓRICO Enriquecer un recorrido del cliente (Customer Journey Map o CJM) con datos de comportamiento significa pasar de un mapa basado en suposiciones o intuiciones a uno fundamentado en evidencia real. Los datos de comportamiento son toda la información generada por las acciones concretas de los usuarios: páginas visitadas, tiempo de permanencia, clics, tasas de abandono, secuencias de navegación, interacciones con emails, conversiones y micro-conversiones. Estos datos provienen de herramientas como Google Analytics, Hotjar, Mixpanel, registros de CRM o plataformas de automatización de marketing. El modelo mental clave es el siguiente: un CJM sin datos es una hipótesis; un CJM con datos es una herramienta de decisión. La diferencia es crítica cuando se trata de optimizar estrategias digitales reales. Errores frecuentes: confundir datos demográficos (quién es el usuario) con datos de comportamiento (qué hace el usuario); utilizar únicamente datos cuantitativos sin cruzarlos con datos cualitativos como entrevistas o encuestas; asumir que un comportamiento observado en un segmento es válido para todos los perfiles de usuario. Mini-ejemplo: una tienda online observa en Google Analytics que el 65% de los usuarios abandona el carrito en la página de pago. Este dato, cruzado con el mapa de recorrido del cliente, permite identificar una fricción concreta en la etapa de 'decisión de compra' y proponer una solución específica (simplificar el formulario, añadir un sello de seguridad, ofrecer más métodos de pago). --- ENUNCIADO DEL TALLER Este taller tiene como objetivo poner en práctica tu capacidad de enriquecer un Customer Journey Map existente utilizando...
