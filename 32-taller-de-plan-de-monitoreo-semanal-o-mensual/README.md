# Taller de : plan de monitoreo semanal o mensual

RECORDATORIO TEÓRICO Un plan de monitoreo es un documento estructurado que define qué KPI se van a seguir, con qué frecuencia, quién es el responsable de su lectura y qué umbrales de alerta deben activar una acción correctiva. Sirve como columna vertebral operativa de cualquier estrategia de marketing digital: sin él, los datos existen pero no se consultan de forma sistemática, y las decisiones se toman tarde o de forma reactiva. Existen dos grandes cadencias de monitoreo: la semanal, recomendada para KPI líderes o indicadores de actividad (clics, tasa de apertura de emails, gasto publicitario diario), y la mensual, más apropiada para KPI rezagados o estratégicos (tasa de conversión global, coste de adquisición de cliente, retorno sobre inversión). Mezclar ambas cadencias en un único plan es un error frecuente que genera ruido y fatiga de análisis. Otros errores habituales: monitorear demasiados KPI sin jerarquía, no asignar un responsable claro para cada indicador, o fijar umbrales de alerta sin referencia a una línea de base real. Un mini-ejemplo: para un e-commerce, el CTR de los anuncios se revisa cada semana (KPI líder), mientras que el ROAS (retorno sobre inversión publicitaria) se analiza mensualmente (KPI rezagado). Cada uno tiene su umbral y su responsable definidos.
